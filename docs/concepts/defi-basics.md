# DeFi 基礎

## 什麼是 DeFi？

**DeFi（Decentralized Finance）= 去中心化金融**

不用銀行、不用信任中介，用智能合約就能：
- 借貸
- 交易
- 賺取收益
- 保險
- 衍生品交易

## DeFi vs 傳統金融

| | **傳統金融** | **DeFi** |
|---|---|---|
| **開戶** | 需要身分證明、審核 | 只需要錢包 |
| **營業時間** | 週一至週五 9-5 | 24/7/365 |
| **誰控制** | 銀行、政府 | 智能合約 |
| **透明度** | 黑箱 | 所有交易公開可查 |
| **權限** | 銀行可以凍結 | 沒有人能阻止 |
| **收益率** | 0.5-2% | 2-20%+（但有風險）|

## 主要 DeFi 類別

### 1. DEX（去中心化交易所）
不用註冊、不用 KYC，直接錢包對錢包交易

| 協議 | 鏈 | 特點 |
|------|----|------|
| Uniswap | Ethereum | AMM 鼻祖 |
| Curve | Ethereum | 穩定幣交換最優 |
| dYdX | 多鏈 | 衍生品交易 |
| Jupiter | Solana | Solana 上最大 DEX |

### 2. 借貸協議
抵押資產借出其他資產，或存入賺取利息

| 協議 | 特點 |
|------|------|
| Aave | 多鏈、閃電貸、機構級 |
| Compound | 最早期的借貸協議 |
| MakerDAO | DAI 穩定幣發行 |

### 3. 收益農場（Yield Farming）
提供流動性或質押代幣賺取收益

| 類型 | 說明 | 風險 |
|------|------|------|
| 流動性挖礦 | 提供交易對賺手續費 | 無常損失 |
| 質押獎勵 | 鎖定代幣賣利息 | 代幣價格下跌 |
| 保險庫策略 | 自動複利優化 | 合約風險 |

### 4. 穩定幣（Stablecoins）
錨定美元的加密貨幣，DeFi 的基礎

| 穩定幣 | 類型 | 市值排名 |
|--------|------|---------|
| USDT | 法幣抵押 | #1 |
| USDC | 法幣抵押 | #2 |
| DAI | 加密抵押 | #3 |

## DeFi 收益來源

1. **借貸利息** — 別人借你的資產付的利息
2. **交易手續費** — DEX 上交易者付的費用
3. **代幣獎勵** — 協議為了吸引流動性發放的代幣
4. **治理收益** — 參與協議治理獲得的獎勵

## DeFi 風險

### ⚠️ 智能合約風險
- 程式碼有 bug 可能導致資金被盜
- 即使是審計過的合約也可能被駭

### ⚠️ 無常損失（Impermanent Loss）
- 提供流動性時，代幣價格變動造成損失
- 賺的手續費可能彌補不了損失

### ⚠️ 清算風險
- 借貸時抵押品價值下跌
- 被強制清算還要付罰金

### ⚠️ 預言機操控
- 價格餵送被操控導致錯誤清算

### ⚠️ Rug Pull
- 協議方捲款跑路

## 新手入門建議

1. **先從大協議開始**
   - Aave、Compound、Uniswap
   - 運作時間長、TVL 高、經過考驗

2. **小額測試**
   - 了解流程後再投入大額
   - 測試存款、提款、借貸

3. **計算真實收益**
   - APY 會變動
   - 考慮 Gas 費用
   - 減去無常損失

4. **分散風險**
   - 不要把所有資金放在一個協議
   - 不要把所有資金放在一條鏈
