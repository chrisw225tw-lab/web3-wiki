# 區塊鏈基礎

## 一句話解釋

**區塊鏈 = 分散式帳本**，所有人一起維護，沒有人能單獨篡改。

## 區塊鏈是什麼？

想像一本**公開的帳本**：
- 每頁記錄一段時間的所有交易
- 每頁（區塊）連到上一頁
- 成千上萬台電腦同時保存這本帳本
- 要改一筆紀錄 = 同時改掉 51% 的電腦

```
區塊 1 → 區塊 2 → 區塊 3 → 區塊 4 → ...
  ↑         ↑         ↑         ↑
創世區塊   包含     包含      包含
         區塊1    區塊2     區塊3
         的雜湊   的雜湊    的雜湊
```

## 核心特性

| 特性 | 說明 |
|------|------|
| **去中心化** | 沒有單一控制者 |
| **透明** | 所有交易公開可查 |
| **不可篡改** | 寫入後無法修改 |
| **抗審查** | 沒有人能阻止交易 |

## 共識機制（Consensus）

區塊鏈如何決定「哪條帳本是對的」？

### Proof of Work（PoW / 工作量證明）
- **原理**：用電腦算力解數學題，先解出來的人獲得記帳權
- **代表**：Bitcoin
- **優點**：極度安全、去中心化
- **缺點**：耗電、速度慢

### Proof of Stake（PoS / 權益證明）
- **原理**：抵押代幣來獲得記帳權，押越多機會越大
- **代表**：Ethereum（2022 年轉型後）
- **優點**：省電、速度快
- **缺點**：可能導致富者更富

## 主要公鏈

| 鏈 | 代幣 | 類型 | 特點 |
|----|------|------|------|
| **Bitcoin** | BTC | PoW | 數位黃金，最去中心化 |
| **Ethereum** | ETH | PoS | 智能合約平台，DeFi 中心 |
| **Solana** | SOL | PoS+ | 高速低費，但較中心化 |
| **BNB Chain** | BNB | PoS | 幣安生態，便宜 |
| **Arbitrum** | ETH | L2 | 以太坊擴容方案 |
| **Base** | ETH | L2 | Coinbase 支持 |

## Layer 1 vs Layer 2

| 層級 | 說明 | 例子 |
|------|------|------|
| **L1（主鏈）** | 基礎區塊鏈 | Bitcoin, Ethereum |
| **L2（擴容層）** | 建在 L1 上的加速層 | Arbitrum, Optimism, Base |

L2 的邏輯：在 L2 做大量交易，最後只把結果寫回 L1，又快又便宜。

## 區塊鏈三角困境（Blockchain Trilemma）

一條鏈**不可能同時**做到：
1. **去中心化**
2. **安全性**
3. **可擴展性**（速度快）

只能選兩個。不同鏈有不同取捨：
- Bitcoin：去中心化 + 安全，但慢
- Solana：安全 + 快，但較中心化

> 💡 **Nakamoto 補充**>

> 答案是：**看你要做什麼。**>
