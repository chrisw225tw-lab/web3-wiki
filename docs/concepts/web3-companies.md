# Web3 知名公司與機構

## 交易所（CEX）

### 🟡 Binance（幣安）
| 項目 | 內容 |
|------|------|
| **成立** | 2017 年，馬爾他註冊 |
| **創辦人** | CZ（趙長鵬）|
| **地位** | 全球最大交易所（交易量）|
| **特色** | 幣種最多、功能最完整、手續費最低 |
| **爭議** | 2023 年 CZ 辭職並認罪，罰款 $43 億 |
| **代幣** | BNB |

**服務**：現貨、合約、理財、Launchpad、NFT 市場

## 公司類型總結

| 類型 | 代表 | 功能 |
|------|------|------|
| **交易所** | Binance、Coinbase、Crypto.com | 買賣加密貨幣 |

| **傳統金融** | BlackRock、Fidelity | ETF、機構服務 |
