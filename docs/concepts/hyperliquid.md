# Hyperliquid

## 什麼是 Hyperliquid？

**Hyperliquid = 高效能 L1 + 頂級衍生品 DEX**

2024 年最現象級的新鏈，結合了：
- Solana 的速度（每秒 200,000+ 訂單）
- CEX 的用戶體驗
- DEX 的自託管安全性

## 核心產品

### 1️⃣ Hyperliquid DEX（衍生品交易）
| 項目 | 內容 |
|------|------|
| **類型** | 永續合約（Perpetuals）|
| **槓桿** | 最高 50 倍 |
| **費用** | 吃單 0.035% / 掛單 0.01% |
| **速度** | 200,000+ 訂單/秒 |
| **特色** | 訂單簿模式（非 AMM）|

**為什麼交易員愛用**：
- 比 Binance 更快、更便宜
- 完全鏈上透明
- 無 KYC
- 支援 100+ 交易對

### 2️⃣ HyperEVM
| 項目 | 內容 |
|------|------|
| **定位** | 相容 Ethereum 的執行層 |
| **用途** | 讓 DeFi 協議建在 Hyperliquid 上 |
| **特色** | 共享訂單簿流動性 |

### 3️⃣ HIP（Hyperliquid Improvement Proposals）
- 社群治理機制
- HYPE 代幣持有者投票

## HYPE 代幣

| 項目 | 內容 |
|------|------|
| **用途** | 治理、質押、手續費折扣 |
| **分配** | 空投給早期用戶（無預售、無 VC）|
| **質押** | 可質押獲得收益 |
| **通縮** | 部分手續費用於回購銷毀 |

**2024 空投傳奇**：
- 早期用戶最高獲得 $100K+ 空投
- 完全基於交易量分配
- 沒有預售、沒有 VC、公平發射

## 技術架構

```
Hyperliquid L1
    ├── 專有共識（HyperBFT）
    ├── 訂單簿引擎（鏈上）
    └── HyperEVM（相容 Ethereum）
```

**HyperBFT**：
- 自研共識機制
- 極致優化交易性能
- 4 個驗證者（目前較中心化）

## 為什麼 Hyperliquid 這麼紅？

✅ **產品好**：真的比 CEX 好用
✅ **公平發射**：沒有 VC 套現壓力
✅ **社群驅動**：用戶有歸屬感
✅ **收入真實**：手續費收入高且持續
✅ **空投預期**：持續有活動獎勵

## 風險

⚠️ **中心化程度高**：只有 4 個驗證者
⚠️ **團隊匿名**：創辦人 @chameleon 身份不明
⚠️ **監管風險**：無 KYC 衍生品可能面臨監管
⚠️ **競爭激烈**：dYdX、GMX、CEX 都是對手
⚠️ **新鏈風險**：運作時間短，未經大考驗

## 如何使用？

1. **連接錢包**：支援 MetaMask、Rabby
2. **存入 USDC**：從 Arbitrum 橋接（官方橋接器）
3. **開始交易**：永續合約、現貨都有
4. **質押 HYPE**：獲得額外收益和空投權重
