# 穩定幣（Stablecoins）

## 什麼是穩定幣？

**穩定幣 = 錨定法幣（通常是美元）的加密貨幣**

解決 Crypto 的大問題：
- ❌ 比特幣一天跌 20%
- ❌ 無法日常支付
- ❌ 進出金麻煩

穩定幣提供：
- ✅ 價格穩定（1 穩定幣 ≈ 1 美元）
- ✅ 24/7 全球轉帳
- ✅ 進入 DeFi 的入口

## 穩定幣類型

### 1️⃣ 法幣抵押型（Fiat-backed）
| 項目 | 內容 |
|------|------|
| **原理** | 每發行 1 穩定幣，銀行存 1 美元 |
| **代表** | USDT、USDC |
| **優點** | 最簡單、最穩定 |
| **風險** | 發行機構倒閉、監管凍結 |

#### USDT（Tether）
| 項目 | 內容 |
|------|------|
| **發行商** | Tether（香港公司）|
| **市值** | $140B+（穩定幣第一）|
| **透明度** | 定期審計，但曾經被質疑 |
| **使用** | 最多交易所支援 |
| **風險** | 監管風險、儲備透明度 |

#### USDC（USD Coin）
| 項目 | 內容 |
|------|------|
| **發行商** | Circle（美國公司，Coinbase 合資）|
| **市值** | $50B+ |
| **透明度** | 每月審計，最受監管認可 |
| **合規** | 美國監管最友善 |
| **風險** | 可被凍結（黑名單地址）|

---

### 2️⃣ 加密抵押型（Crypto-backed）
| 項目 | 內容 |
|------|------|
| **原理** | 抵押加密資產（如 ETH），超額抵押生成穩定幣 |
| **代表** | DAI、crvUSD |
| **優點** | 去中心化、抗審查 |
| **風險** | 清算風險、智能合約風險 |

#### DAI（MakerDAO）
| 項目 | 內容 |
|------|------|
| **機制** | 抵押 ETH、WBTC 等生成 DAI |
| **抵押率** | 通常 150%+（超額抵押）|
| **治理** | MKR 代幣持有者治理 |
| **特色** | 最去中心化的主流穩定幣 |
| **風險** | 如果抵押品暴跌，可能系統性風險 |

**例子**：
- 存入 $150 ETH → 借出 $100 DAI
- ETH 價格跌太多 → 被強制清算

---

### 3️⃣ 演算法型（Algorithmic）
| 項目 | 內容 |
|------|------|
| **原理** | 用演算法調節供需，維持錨定 |
| **代表** | UST（已崩盤）、FRAX |
| **優點** | 完全去中心化、資本效率高 |
| **風險** | **極高**，可能死亡螺旋 |

#### UST 崩盤事件（2022）
| 時間 | 事件 |
|------|------|
| **巔峰** | UST $18B 市值，Anchor 協議 20% APY |
| **5/7** | UST 開始脫錨（$0.98）|
| **5/9** | 暴跌至 $0.35 |
| **5/13** | 幾乎歸零（$0.09）|
| **結果** | $40B 市值蒸發，LUNA 歸零 |

**教訓**：演算法穩定幣是實驗，風險極高。

## 穩定幣收益

### 為什麼能賺利息？
穩定幣發行商把儲備拿去買美債：
- 美國國庫券 4-5% 收益
- 分一部分給用戶

### 收益來源
| 方式 | APY | 風險 |
|------|-----|------|
| Aave/Compound 借貸 | 3-8% | 智能合約風險 |
| 質押 USDe（Ethena）| 5-15% | 新機制風險 |
| 流動性挖礦 | 10-50% | 無常損失、Rug Pull |
| CEX 理財 | 2-5% | 交易所風險 |
