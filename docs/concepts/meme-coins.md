# Meme 幣（Meme Coins）

## 什麼是 Meme 幣？

**Meme 幣 = 基於迷因文化的加密貨幣**

沒有「基本面」：
- ❌ 沒有產品
- ❌ 沒有收入
- ❌ 沒有技術創新

但有「文化面」：
- ✅ 社群共識
- ✅ 迷因傳播力
- ✅ 投機熱情

## 為什麼 Meme 幣會漲？

| 因素 | 說明 |
|------|------|
| **社群 FOMO** | 怕錯過下一個 Doge |
| **KOL 喊单** | 網紅、交易員推廣 |
| **交易所上幣** | Binance、Coinbase 上線 = 暴漲 |
| **迷因傳播** | Twitter、TikTok 病毒式擴散 |
| **賭場心理** | 「反正便宜，買一點試試」|
| **做空擠壓** | 做空的人被迫買回，推升價格 |

## Meme 幣的風險

### ⚠️ 99.9% 會歸零
| 數據 | 說明 |
|------|------|
| **存活率** | 只有 0.1% Meme 幣能活過 1 年 |
| **拉盤砸盤** | 創辦人/早期買家高點拋售 |
| **無限供應** | 很多 Meme 幣持續增發，稀釋價值 |
| **流動性消失** | 熱度過後，賣不掉 |

### ⚠️ 常見詐騙
| 手法 | 說明 |
|------|------|
| **Rug Pull** | 創辦人捲款跑路 |
| **蜜罐（Honeypot）** | 只能買不能賣 |
| **假合約** | 名字相似，合約是假的 |
| **狙击機器人** | 開盤瞬間被機器人買光，散戶買在高點 |

## 如何「玩」Meme 幣？（如果你堅持要玩）

### ✅ 相對安全的做法
1. **只投入你能完全損失的錢** — 當作買彩券
2. **分散多個** — 不要 all in 一個
3. **設停損** — 跌 50% 就砍，不要凹單
4. **看鏈上數據** — 
   - [DexScreener](https://dexscreener.com) 看價格
   - [Birdeye](https://birdeye.so) 看 Solana 幣
   - [Token Sniffer](https://tokensniffer.com) 查合約

### ❌ 絕對不要做
- 借錢買 Meme 幣
- 把全部家當壓一個幣
- 相信「這個一定會上幣安」
- 追高殺低（FOMO 買高點）

## Meme 幣的「投資框架」（認真的）

雖然荒謬，但專業玩家確實有策略：

| 評估項目 | 說明 |
|---------|------|
| **社群熱度** | Twitter 討論度、搜尋趨勢 |
| **持幣分布** | 是否集中在少數人手中 |
| **流動性** | 池子深度夠不夠 |
| **敘事強度** | 迷因是否有文化根基 |
| **交易所預期** | 是否有機會上大所 |
| **時間窗口** | 通常只有 1-4 週黃金期 |
