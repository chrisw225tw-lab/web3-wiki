# NFT 基礎

## 什麼是 NFT？

**NFT（Non-Fungible Token）= 非同質化代幣**

- **非同質化** = 每個都獨一無二，不能互換
- 相對於 **FT（Fungible Token）** = 同質化代幣，可以互換（如 ETH、USDC）

## 比喻理解

| 同質化 | 非同質化 |
|--------|---------|
| 鈔票（每張 100 元都一樣）| 房子（每間都不同）|
| 比特幣（每個 BTC 相同）| 藝術品（每幅都不同）|
| 股票（同公司每張相同）| 遊戲裝備（每個屬性不同）|

## NFT 能做什麼？

### 1. 數位藝術收藏
- PFP（頭像項目）：CryptoPunks、Bored Ape
- 數位藝術：Beeple 的作品賣了 $6900 萬

### 2. 遊戲資產
- 武器、角色、土地
- 真正擁有，可以交易
- 例：Axie Infinity、The Sandbox

### 3. 會員資格
- 進入特定社群的通行證
- 線下活動門票
- 例：Bored Ape Yacht Club

### 4. 身份與憑證
- 學歷證書
- 活動參與證明
- 職業認證

### 5. 實體資產上鏈
- 房地產所有權
- 奢侈品認證
- 供應鏈追蹤

## 如何運作？

```
NFT 合約
    ├── Token ID（唯一編號）
    ├── Metadata（元數據）
    │       ├── 名稱
    │       ├── 描述
    │       ├── 圖片 URL
    │       └── 屬性
    └── 所有權記錄
```

## 購買 NFT 須知

### ✅ 買之前檢查
- [ ] 合約地址正確（防假貨）
- [ ] 查看交易歷史
- [ ] 了解版稅費用
- [ ] 確認是真品（Verified）

### ⚠️ 常見風險
- **Rug Pull**：項目方捲款消失
- **版權問題**：買了 NFT 不一定擁有商業權利
- **流動性風險**：可能賣不掉
- **價格波動**：NFT 價格可能歸零

## 版稅（Royalty）

創作者可以在每次轉售時收取版稅：
- 通常 2.5-10%
- 自動由合約執行
- 讓創作者持續受益

但注意：有些平台開始讓買家選擇是否付版稅
