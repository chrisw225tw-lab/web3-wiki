# 什麼是 Web3？

## 一句話解釋

**Web3 = 去中心化的網際網路**，使用者真正擁有自己的資料和資產。

## Web1 → Web2 → Web3

| 世代 | 時期 | 特點 | 代表 |
|------|------|------|------|
| **Web1** | 1990s-2000s | 靜態網頁，只能閱讀 | Yahoo, 早期網站 |
| **Web2** | 2000s-現在 | 互動、社交、使用者創作內容 | Facebook, Google, YouTube |
| **Web3** | 現在-未來 | 去中心化、用戶擁有資料、代幣經濟 | Ethereum, DeFi, NFT |

## Web3 的核心特性

### 1. 去中心化（Decentralization）
- 沒有單一公司控制一切
- 資料存在區塊鏈上，分散在全球節點
- 沒有人能隨便關掉你的帳號

### 2. 無需許可（Permissionless）
- 任何人都能參與
- 不需要銀行批准就能轉帳
- 不需要平台批准就能發布內容

### 3. 無需信任（Trustless）
- 不需要相信任何中間人
- 數學和密碼學保證安全
- 程式碼就是法律（Code is Law）

### 4. 可組合性（Composability）
- 像樂高一樣，不同協議可以組合
- 在 A 協議的資產可以在 B 協議使用

## 你可能已經用過的 Web3

- **MetaMask** — 加密錢包
- **OpenSea** — NFT 交易平台
- **Uniswap** — 去中心化交易所
- **Axie Infinity** — 鏈上遊戲
