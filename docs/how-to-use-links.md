# 如何使用雙向連結

## Obsidian 式語法

這個 Wiki 支援 `[[術語名稱]]` 語法，自動連結到定義頁面。

## 範例

當你在任何文章裡寫：

```markdown
如果你想買加密貨幣，可以使用 [[CEX]] 如幣安或 Coinbase。
但記得，長期持有應該使用 [[Ledger]] 或 [[Trezor]] 這類[[硬體錢包]]。

玩[[衍生品]]要小心，95% 散戶會虧錢。
還要當心 [[VC套現]]，代幣解鎖時可能暴跌。
```

顯示效果會自動變成可點擊的連結：

> 如果你想買加密貨幣，可以使用 [[CEX]] 如幣安或 Coinbase。
> 但記得，長期持有應該使用 [[Ledger]] 或 [[Trezor]] 這類[[硬體錢包]]。
> 
> 玩[[衍生品]]要小心，95% 散戶會虧錢。
> 還要當心 [[VC套現]]，代幣解鎖時可能暴跌。

## 如何新增術語頁面

1. 在 `docs/terms/` 創建新檔案，例如 `new-term.md`
2. 寫入術語定義
3. 更新 `mkdocs.yml` 導航
4. 現在可以在任何文章用 `[[new-term]]` 連結

## 注意事項

- 檔名會自動轉成連結（大小寫不敏感）
- 如果頁面不存在，連結會顯示為紅色或無效
- 可以用 `[[術語|顯示文字]]` 自訂顯示文字（部分插件支援）
