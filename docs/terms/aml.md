# AML（反洗錢）

## 定義

**AML = Anti-Money Laundering（反洗錢）**

防止非法資金通過金融系統洗白的法規和措施。在加密貨幣領域，AML 要求交易所和其他服務提供商識別並報告可疑交易。

## AML 的核心措施

| 措施 | 說明 |
|------|------|
| **KYC** | 了解你的客戶，驗證身份 |
| **交易監控** | 追蹤大額和可疑交易 |
| **風險評估** | 評估客戶和交易的風險等級 |
| **報告義務** | 向監管機構報告可疑活動 |

## 為什麼重要

✅ **合法合規**
- 符合各國法律法規要求
- 避免平台被罰款或關閉

✅ **保護生態**
- 減少非法資金流入加密市場
- 提升行業整體形象

✅ **個人保護**
- 避免無意中參與洗錢活動
- 保護自己的資產安全

## 對用戶的影響

⚠️ **交易所要求**
- 註冊時必須完成 [[KYC]]
- 可能需要提供收入來源證明
- 大額提款可能需要額外審核

⚠️ **DeFi 的灰色地帶**
- 去中心化交易所通常無 AML
- 但仍可能通過鏈上分析追蹤
- Tornado Cash 等隱私工具受監管關注

## 常見觸發 AML 審查的情況

| 情況 | 風險等級 |
|------|---------|
| 單筆大額交易（>10萬美元）| ⚠️ 中等 |
| 頻繁的小額存款後大額提款 | 🔴 高 |
| 與已知黑地址有交互 | 🔴 高 |
| 短時間內多國 IP 登入 | 🟡 低-中 |

## 注意事項

⚠️ **不要嘗試規避 AML**
- 使用他人身份註冊可能觸犯法律
- 混幣器使用可能留下永久記錄
- 鏈上交易是公開且不可篡改的

⚠️ **保持記錄**
- 保存交易記錄和資金來源證明
- 有需要時能向交易所或監管機構說明
