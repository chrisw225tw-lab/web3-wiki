# 硬體錢包（Hardware Wallets）

## 定義

**硬體錢包 = 離線儲存私鑰的物理設備**

私鑰永遠不離開設備，即使連接被駭電腦也安全。

## 為什麼要用硬體錢包？

| 場景 | 風險 | 硬體錢包保護 |
|------|------|-------------|
| 電腦中病毒 | 私鑰被竊 | ✅ 私鑰在設備內 |
| 釣魚網站 | 誘導簽名 | ✅ 需在設備上確認 |
| 手機錢包被駭 | 資產被轉走 | ✅ 無法遠端操作 |
| 忘記助記詞 | 資產永久丟失 | ⚠️ 仍需備份助記詞 |

## 冷錢包 vs 熱錢包

| | 硬體錢包（冷）| 軟體錢包（熱）|
|---|---|---|
| **連網** | ❌ 平時離線 | ✅ 連網 |
| **安全** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| **方便** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **價格** | $60-250 | 免費 |
| **適合** | 大額長期 | 日常小額 |

## 使用建議

### 資產配置
| 金額 | 建議 |
|------|------|
| < $1,000 | 熱錢包足夠 |
| $1,000 - $10,000 | 考慮買硬體錢包 |
| > $10,000 | 強烈建議硬體錢包 |
| > $100,000 | 多個硬體錢包分散 |

### 購買注意
⚠️ **絕對不要**：
- 從非官方渠道買（Amazon 第三方、淘寶）
- 買二手/開封過的
- 使用別人給的「預設助記詞」

✅ **一定要**：
- 官網購買，檢查封條
- 首次使用時初始化（產生新助記詞）
- 多份助記詞備份
