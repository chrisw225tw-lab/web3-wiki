# 區塊鏈術語表

## 基礎概念

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Address** | 地址 | 區塊鏈上的帳號，用於收款，公開無妨 |
| **Block** | 區塊 | 一批交易的集合，包含時間戳和前一個區塊的雜湊 |
| **Blockchain** | 區塊鏈 | 由區塊串連而成的分散式帳本 |
| **Consensus** | 共識機制 | 網路節點達成一致的規則（如 PoW、PoS）|
| **Cryptography** | 密碼學 | 保護資訊安全的數學技術 |
| **Decentralization** | 去中心化 | 沒有單一控制點，權力分散 |
| **Distributed Ledger** | 分散式帳本 | 多個節點同時維護的帳本 |
| **Hash** | 雜湊 | 將任意資料轉為固定長度字串的函數 |
| **Immutable** | 不可篡改 | 一旦寫入就無法更改 |
| **Node** | 節點 | 運行區塊鏈軟體的電腦 |
| **Peer-to-Peer (P2P)** | 點對點 | 直接連接，無需中介 |
| **Private Key** | 私鑰 | 證明資產所有權的機密密碼 |
| **Public Key** | 公鑰 | 由私鑰衍生，可公開 |
| **Transaction** | 交易 | 區塊鏈上的資產轉移紀錄 |
| **Wallet** | 錢包 | 管理私鑰、與區塊鏈互動的工具 |

## 共識機制

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Proof of Work (PoW)** | 工作量證明 | 用算力競爭記帳權，Bitcoin 使用 |
| **Proof of Stake (PoS)** | 權益證明 | 用抵押代幣獲得記帳權，Ethereum 使用 |
| **Validator** | 驗證者 | PoS 中負責驗證交易的節點 |
| **Miner** | 礦工 | PoW 中用算力挖礦的節點 |
| **Staking** | 質押 | 鎖定代幣以支持網路並獲得獎勵 |
| **Slashing** | 削減 | 對作惡驗證者的懲罰（沒收質押）|

## 網路結構

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Layer 1 (L1)** | 第一層 | 基礎區塊鏈（如 Ethereum、Bitcoin）|
| **Layer 2 (L2)** | 第二層 | 建立在 L1 上的擴容方案（如 Arbitrum）|
| **Sidechain** | 側鏈 | 與主鏈平行運行的獨立鏈 |
| **Bridge** | 跨鏈橋 | 在不同鏈之間轉移資產的協議 |
| **Mainnet** | 主網 | 正式運行的區塊鏈網路 |
| **Testnet** | 測試網 | 供開發測試用的網路，代幣無價值 |
| **Genesis Block** | 創世區塊 | 區塊鏈的第一個區塊 |

## 代幣類型

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Token** | 代幣 | 區塊鏈上的數位資產 |
| **Coin** | 幣 | 原生的區塊鏈資產（如 BTC、ETH）|
| **ERC-20** | ERC-20 標準 | Ethereum 上的同質化代幣標準 |
| **ERC-721** | ERC-721 標準 | Ethereum 上的 NFT 標準 |
| **Altcoin** | 山寨幣 | Bitcoin 以外的所有加密貨幣 |
| **Stablecoin** | 穩定幣 | 錨定法幣（如 USD）的代幣 |
| **Governance Token** | 治理代幣 | 用於投票決定協議走向的代幣 |
| **Utility Token** | 功能型代幣 | 在特定生態系中使用的代幣 |
