# 交易術語表

## 基礎交易

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Buy/Sell Order** | 買/賣單 | 下單購買或出售資產 |
| **Market Order** | 市價單 | 以當前市場價格立即成交 |
| **Limit Order** | 限價單 | 設定特定價格，達到才成交 |
| **Stop Loss** | 止損 | 價格跌到某水平自動賣出，限制虧損 |
| **Take Profit** | 止盈 | 價格漲到某水平自動賣出，鎖定利潤 |
| **Liquidity** | 流動性 | 資產能快速買賣而不影響價格的程度 |
| **Volume** | 成交量 | 一定時間內的交易總量 |
| **Volatility** | 波動性 | 價格變動的幅度和頻率 |
| **Spread** | 買賣價差 | 最高買價和最低賣價的差距 |

## 交易所相關

| 英文 | 中文 | 解釋 |
|------|------|------|
| **CEX** | 中心化交易所 | 由公司運營的交易平台（如幣安）|
| **DEX** | 去中心化交易所 | 無需信任的點對點交易（如 Uniswap）|
| **Order Book** | 訂單簿 | 記錄所有買賣報價的清單 |
| **AMM** | 自動做市商 | 用數學公式定價的 DEX 機制 |
| **Liquidity Pool** | 流動性池 | 用戶存入的資金池，供 AMM 交易使用 |
| **LP Token** | 流動性提供者代幣 | 證明你在流動性池中份額的代幣 |
| **Slippage** | 滑點 | 實際成交價格與預期的差異 |
| **Impermanent Loss** | 無常損失 | 提供流動性時因價格變動造成的潛在損失 |

## 市場狀態

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Bull Market** | 牛市 | 價格持續上漲的市場 |
| **Bear Market** | 熊市 | 價格持續下跌的市場 |
| **ATH** | 歷史最高價 | All-Time High |
| **ATL** | 歷史最低價 | All-Time Low |
| **Support** | 支撐位 | 價格難以跌破的水平 |
| **Resistance** | 阻力位 | 價格難以突破的水平 |
| **Correction** | 回調 | 價格從高點下跌 10-20% |
| **Crash** | 崩盤 | 價格快速大幅下跌 |
| **Pump** | 拉盤 | 價格快速上漲 |
| **Dump** | 砸盤 | 價格快速下跌 |

## 交易者類型

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Whale** | 巨鯨 | 持有大量加密貨幣的投資者 |
| **Retail Investor** | 散戶 | 一般小額投資者 |
| **Day Trader** | 日內交易者 | 當天開平倉的短線交易者 |
| **Swing Trader** | 波段交易者 | 持有數天到數週的交易者 |
| **HODLer** | 長期持有者 | 買入後長期不賣的投資者 |
| **Ape** | 盲目跟單者 | 沒做研究就衝進去買的人 |

## 市場情緒

| 英文 | 中文 | 解釋 |
|------|------|------|
| **FOMO** | 錯失恐懼症 | Fear Of Missing Out，怕錯過上漲而衝動買入 |
| **FUD** | 恐懼不確定懷疑 | Fear Uncertainty Doubt，散播負面消息 |
| **Diamond Hands** | 鑽石手 | 無論如何都不賣的堅定持有者 |
| **Paper Hands** | 紙手 | 價格一跌就恐慌賣出的人 |
| **Moon** | 暴漲 | 「To the moon!」預期價格大漲 |
| **Rekt** | 被坑慘 | Wrecked，虧大錢或被清算 |
| **WAGMI** | 我們都會成功 | We Are Gonna Make It |
| **NGMI** | 不會成功 | Not Gonna Make It |
