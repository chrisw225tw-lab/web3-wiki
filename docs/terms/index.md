# 術語索引

這裡收錄 Web3 最常用的術語，按主題分類：

| 分類 | 內容 |
|------|------|
| [區塊鏈術語](blockchain.md) | 共識、雜湊、節點等技術概念 |
| [交易術語](trading.md) | 訂單簿、滑點、無常損失等 |
| [DeFi 術語](defi.md) | 流動性挖礦、收益農場、閃電貸等 |
| [安全術語](security.md) | 釣魚、Rug Pull、蜜罐等風險 |

## 最常見 20 個術語速查

| 英文 | 中文 | 一句話解釋 |
|------|------|-----------|
| **Address** | 地址 | 你的收款帳號，公開沒問題 |
| **Blockchain** | 區塊鏈 | 分散式、不可篡改的帳本 |
| **DeFi** | 去中心化金融 | 沒有銀行的金融服務 |
| **DEX** | 去中心化交易所 | 點對點直接交易，不用信任平台 |
| **Gas** | 燃料費 | 執行交易的手續費 |
| **HODL** | 長期持有 | Hold 的錯字，變成術語 |
| **NFT** | 非同質化代幣 | 獨一無二的數位資產 |
| **Private Key** | 私鑰 | 絕對不能給任何人的密碼 |
| **Seed Phrase** | 助記詞 | 12/24 個單字，可還原私鑰 |
| **Smart Contract** | 智能合約 | 自動執行的區塊鏈程式 |
| **Token** | 代幣 | 區塊鏈上的數位資產 |
| **Wallet** | 錢包 | 管理私鑰、收發資產的工具 |
| **DYOR** | 自己做研究 | Do Your Own Research |
| **FOMO** | 錯失恐懼症 | Fear Of Missing Out，怕踏空 |
| **FUD** | 恐懼不確定懷疑 | Fear Uncertainty Doubt |
| **Whale** | 巨鯨 | 持有大量加密貨幣的人 |
| **Ape in** | 盲目投入 | 沒做研究就 all in |
| **Rug Pull** | 捲款跑路 | 項目方突然捲走所有資金 |
| **Moon** | 暴漲 | 「To the moon!」飛向月球 |
| **Rekt** | 被坑慘 | Wrecked，虧大錢或被駭 |
