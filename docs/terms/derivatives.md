# 衍生品（Derivatives）

## 定義

**衍生品 = 從「標的資產」派生價值的金融合約**

不直接擁有資產，而是賭資產價格變動。

## 主要類型

### 1️⃣ 永續合約（Perpetuals / Perps）— 最流行

| 項目 | 說明 |
|------|------|
| **原理** | 沒有到期日的合約，用「資金費率」維持錨定 |
| **槓桿** | 1x - 125x |
| **做多** | 賭價格上漲 |
| **做空** | 賭價格下跌 |
| **費用** | 開倉/平倉手續費 + 資金費率 |

**例子**：
- BTC 現在 $100,000
- 你開 10x 多倉，投入 $1,000
- BTC 漲到 $110,000（+10%）
- 你賺 $1,000（本金翻倍）
- BTC 跌到 $99,000（-1%）
- 你被[[清算]]，本金歸零

### 2️⃣ 期貨（Futures）

| 項目 | 說明 |
|------|------|
| **原理** | 約定未來某時間以某價格買賣 |
| **到期日** | 有特定結算時間 |
| **用途** | 對沖風險、投機 |

### 3️⃣ 期權（Options）

| 項目 | 說明 |
|------|------|
| **Call** | 買權（賭上漲）|
| **Put** | 賣權（賭下跌）|
| **權利金** | 買期權時付出的費用 |
| **特點** | 最大損失有限（權利金），最大獲利無限 |

## 衍生品 vs 現貨

| | 現貨 | 衍生品 |
|---|---|-------|
| **擁有資產** | ✅ 有 | ❌ 沒有 |
| **槓桿** | ❌ 無 | ✅ 有 |
| **可做空** | ❌ 不能 | ✅ 可以 |
| **風險** | 價格波動 | 價格波動 + 清算風險 |
| **適合** | 長期持有 | 短線交易、對沖 |

## 風險警告

### ⚠️ 清算風險（Liquidation）

**例子**：
- 你開 10x 多倉
- BTC 跌 10% = 你的保證金歸零
- 交易所自動平倉，你失去全部本金

### ⚠️ 資金費率（Funding Rate）

- 每 8 小時結算一次
- 做多的人多 → 多頭付費給空頭
- 長期持有成本可能很高

### ⚠️ 插針（Wick）

- 價格瞬間劇烈波動
- 即使馬上恢復，高槓桿已被清算
- 交易所「剛好」觸及你的清算價

## 統計數據

> 📊 **95% 的散戶在衍生品交易虧錢**

原因：
1. 過度槓桿
2. 沒有止損
3. 情緒交易（FOMO、恐慌）
4. 不懂資金費率成本

## Nakamoto 的建議

| 經驗等級 | 建議 |
|---------|------|
| 新手 | ❌ 不要碰合約 |
| 有經驗 | ⚠️ 最多 3-5x 槓桿 |
| 專業 | 嚴格止損，控制倉位 |
