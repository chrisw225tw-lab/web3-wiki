# VC 套現（VC Dump）

## 定義

**VC 套現 = 風險投資（Venture Capital）在項目上線後拋售代幣獲利**

## 運作模式

```
VC 投資流程：
1. 項目早期 → VC 以極低價買入代幣（$0.01 或更低）
2. 項目發展 → VC 提供資源、人脈、上交易所關係
3. 代幣上線 → 散戶在交易所高價買入
4. 解鎖期到 → VC 拋售代幣，獲利離場
5. 代幣暴跌 → 散戶被套在高点
```

## 典型解鎖時間表

| 階段 | 時間 | 發生什麼 |
|------|------|---------|
| **TGE** | 上線日 | 釋放 5-10% |
| **Cliff** | 3-12 個月 | 完全不釋放 |
| **線性解鎖** | 之後 1-4 年 | 每月釋放一部分 |
| **大量解鎖** | 特定時間點 | 可能暴跌 |

## 警示信號

⚠️ **代幣分配不均**
- VC + 團隊佔 40-60%
- 散戶只能搶少量

⚠️ **完全稀釋估值（FDV）過高**
- 市值 $1B，FDV $10B
- 代表還有 90% 代幣未釋放

⚠️ **短期大量解鎖**
- 查 [TokenUnlocks](https://token.unlocks.app) 看解鎖時間表

## 如何避免被割？

### 投資前檢查
- [ ] 查看代幣分配（CoinMarketCap → Tokenomics）
- [ ] FDV / 市值比率不要太高
- [ ] 查解鎖時間表
- [ ] 團隊+VC 鎖定期夠長（2 年以上）

### 投資後監控
- [ ] 標記日曆提醒解鎖日期
- [ ] 大量解鎖前考慮減倉
- [ ] 關注 VC 錢包動向（[[鏈上分析]]）

## 正面案例：公平發射

一些項目選擇**沒有 VC** 或**公平分配**：

| 項目 | 做法 | 結果 |
|------|------|------|
| **Bitcoin** | 無預挖、無 VC | 最去中心化 |
| **Hyperliquid** | 全部空投給用戶 | 社群忠誠度高 |
| **PEPE** | 無團隊分配 | 純社群驅動 |
