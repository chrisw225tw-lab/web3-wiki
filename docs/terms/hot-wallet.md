# 熱錢包（Hot Wallet）

## 定義

**熱錢包 = 私鑰儲存在聯網設備上的加密貨幣錢包**

隨時保持互聯網連接，方便快速進行交易。常見形式包括：
- 手機 App（MetaMask、Trust Wallet）
- 瀏覽器插件（MetaMask、Rabby）
- 交易所錢包
- 桌面錢包

## 熱錢包 vs [[冷錢包]]

| 項目 | 熱錢包 | 冷錢包 |
|------|--------|--------|
| **聯網狀態** | 在線 | 離線 |
| **安全性** | 🟡 中等（有被駭風險）| 🟢 極高 |
| **便利性** | 🟢 極高（隨時交易）| 🟡 較低 |
| **交易速度** | 即時 | 需手動操作 |
| **適用場景** | 日常交易、DeFi 互動 | 長期儲存、大額資產 |

## 為什麼重要

✅ **極致便利**
- 隨時隨地發送和接收加密貨幣
- 即時參與 DeFi、NFT 交易
- 一鍵連接各種 DApp

✅ **DeFi 必備**
- 幾乎所有 DeFi 操作都需要熱錢包
- 與智能合約互動的主要工具

✅ **零成本開始**
- 大多數熱錢包完全免費
- 無需購買額外硬件

## 使用建議

| 用途 | 建議 |
|------|------|
| 日常小額交易 | ✅ 適合使用熱錢包 |
| DeFi 操作 | ✅ 必須使用熱錢包 |
| NFT 交易 | ✅ 適合使用熱錢包 |
| 大額長期持有 | ❌ 轉移至[[冷錢包]] |
| 全部資產 | ❌ 不要只使用一個熱錢包 |

## 風險與注意事項

⚠️ **網絡安全威脅**
- 釣魚網站竊取私鑰
- 惡意軟件和鍵盤記錄器
- 偽造的錢包 App

⚠️ **常見攻擊手法**
| 攻擊類型 | 說明 | 防範方法 |
|---------|------|---------|
| 釣魚網站 | 假冒知名 DApp | 仔細檢查網址 |
| 假錢包 App | 應用商店中的仿冒品 | 只從官方渠道下載 |
| 授權盜竊 | 惡意合約獲得無限授權 | 定期檢查並撤銷授權 |
| 社交工程 | 騙取助記詞 | 永遠不分享助記詞 |

⚠️ **保護措施**
1. **只存小額**：將大部分資產放在冷錢包
2. **分開使用**：不同用途使用不同錢包地址
3. **定期檢查**：檢查合約授權，及時撤銷不必要的權限
4. **保持更新**：及時更新錢包軟件到最新版本
