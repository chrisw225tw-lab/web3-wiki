# 安全術語表

## 詐騙類型

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Rug Pull** | 捲款跑路 | 項目方突然帶走所有資金消失 |
| **Pump and Dump** | 拉高出貨 | 散布假消息拉盤後高點拋售 |
| **Phishing** | 釣魚 | 偽裝成官方網站騙取私鑰 |
| **Honey Pot** | 蜜罐 | 只能買不能賣的詐騙代幣 |
| **Exit Scam** | 跑路詐騙 | 籌款後項目方消失 |
| **Social Engineering** | 社交工程 | 利用人性弱點騙取資訊 |
| **Airdrop Scam** | 空投詐騙 | 假空投誘導授權惡意合約 |
| **Impersonation** | 冒充詐騙 | 假扮名人或官方騙取資產 |

## 技術攻擊

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Smart Contract Exploit** | 合約漏洞攻擊 | 利用合程式碼 bug 盜取資金 |
| **Reentrancy Attack** | 重入攻擊 | 遞迴調用合約竊取資金 |
| **Flash Loan Attack** | 閃電貸攻擊 | 用閃電貸操縱價格套利 |
| **Oracle Manipulation** | 預言機操控 | 操控價格餵送數據獲利 |
| **Bridge Hack** | 跨鏈橋被駭 | 攻擊跨鏈橋盜取鎖定資產 |
| **Private Key Leak** | 私鑰洩漏 | 私鑰被盜導致資產損失 |
| **SIM Swap** | SIM 卡交換 | 劫持手機號碼重設帳號密碼 |

## 錢包安全

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Cold Wallet** | 冷錢包 | 不連網的硬體錢包 |
| **Hot Wallet** | 熱錢包 | 連網的軟體錢包 |
| **Hardware Wallet** | 硬體錢包 | 專用設備儲存私鑰 |
| **Multisig** | 多簽錢包 | 需要多把私鑰才能動用資產 |
| **Air-gapped** | 氣隙隔離 | 完全不連網的設備 |
| **Seed Phrase** | 助記詞 | 恢復錢包的 12/24 個單字 |
| **Passphrase** | 密語 | 額外的密碼增強安全 |

## 授權與權限

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Token Approval** | 代幣授權 | 允許合約動用你的代幣 |
| **Unlimited Approval** | 無限授權 | 授權合約動用無限數量代幣 |
| **Revoke** | 撤銷 | 取消已授權的權限 |
| **Permit** | 許可 | 無需 gas 的簽名授權 |
| **Spender** | 被授權方 | 獲得代幣使用權的合約/地址 |

## 警示信號

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Red Flag** | 危險信號 | 需要注意的警告跡象 |
| **Too Good To Be True** | 好到不像真的 | 高得離譜的收益承諾 |
| **Anonymous Team** | 匿名團隊 | 無法查證身分的開發者 |
| **Unaudited Contract** | 未審計合約 | 沒有第三方安全審查 |
| **Copycat Project** | 抄襲項目 | 複製其他項目的程式碼 |
| **Fake Volume** | 假成交量 | 刷出來的交易量 |
| **Honeypot Token** | 蜜罐代幣 | 只能買不能賣的詐騙幣 |

## 安全實踐

| 英文 | 中文 | 解釋 |
|------|------|------|
| **DYOR** | 自己做研究 | Do Your Own Research |
| **2FA** | 雙重認證 | Two-Factor Authentication |
| **Burner Wallet** | 拋棄式錢包 | 用於單一用途的臨時錢包 |
| **Dusting Attack** | 粉塵攻擊 | 發送微量代幣追蹤地址 |
| **Nonce** | 交易序號 | 確保交易唯一性的數字 |
