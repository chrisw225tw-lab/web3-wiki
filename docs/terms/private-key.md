# 私鑰（Private Key）

## 定義

**私鑰 = 控制加密貨幣資產的秘密密碼**

一串由數字和字母組成的長字符串，用於：
- 簽署交易（證明資產所有權）
- 生成對應的公鑰和錢包地址
- 完全控制對應地址的所有資產

```
以太坊私鑰範例（64位十六進制）：
0x4c0883a69102937d6231471b5dbb6204fe5129617082792ae468d01a3f362318
```

## 私鑰 vs 公鑰 vs 地址

| 項目 | 功能 | 是否可以公開 |
|------|------|-------------|
| **私鑰** | 簽署交易、控制資產 | ❌ 絕對保密 |
| **公鑰** | 由私鑰生成，用於加密 | ✅ 可以公開 |
| **地址** | 收款用，類似銀行帳號 | ✅ 可以公開 |

關係：**私鑰 → 公鑰 → 地址**（不可逆）

## 為什麼重要

✅ **資產控制權**
- 擁有私鑰 = 擁有資產
- "Not your keys, not your coins"

✅ **身份驗證**
- 無需密碼或個人信息
- 數學保證的安全性

✅ **去中心化基礎**
- 無需第三方即可管理資產
- 真正實現金融自主

## 私鑰的儲存方式

| 方式 | 安全性 | 便利性 |
|------|--------|--------|
| [[硬件錢包]] | 🟢 極高 | 中等 |
| [[助記詞]]備份 | 🟢 高 | 高 |
| 加密文件 | 🟡 中等 | 高 |
| 紙質備份 | 🟢 高（防駭）| 低（易損壞）|
| 雲端儲存 | 🔴 極低 | 極高 |
| 截圖/拍照 | 🔴 極低 | 高 |

## 風險與注意事項

⚠️ **絕對禁忌**
| 行為 | 後果 |
|------|------|
| 截圖保存私鑰 | 可能被自動上傳雲端 |
| 通過網絡傳輸私鑰 | 被攔截竊取 |
| 輸入私鑰到任何網站 | 幾乎一定是詐騙 |
| 分享私鑰給「客服」| 資產立即被盜 |

⚠️ **常見詐騙話術**
- "我們需要驗證你的私鑰"
- "輸入私鑰來領取空投"
- "技術問題，請提供私鑰協助修復"
- **記住：任何人向你索要私鑰都是騙子！**

⚠️ **丟失即永久損失**
- 沒有「忘記密碼」功能
- 沒有客服可以幫你恢復
- 私鑰丟失 = 資產永遠無法取回

## 安全儲存建議

### ✅ 推薦做法
1. **使用[[助記詞]]**：比原始私鑰更易備份
2. **多份備份**：至少 2-3 份，存放在不同地點
3. **防火防水**：使用金屬助記詞板
4. **離線儲存**：不要以任何電子形式保存

### ❌ 絕對避免
- 郵件、簡訊、通訊軟件
- 雲端硬碟、筆記 App
- 手機相簿、截圖
- 腦記（容易遺忘或混淆）
