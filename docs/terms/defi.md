# DeFi 術語表

## DeFi 基礎

| 英文 | 中文 | 解釋 |
|------|------|------|
| **DeFi** | 去中心化金融 | 建立在區塊鏈上的金融服務 |
| **TradFi** | 傳統金融 | Traditional Finance，指銀行等機構 |
| **TVL** | 總鎖倉價值 | Total Value Locked，協議中鎖定的資產總值 |
| **APY** | 年化收益率 | Annual Percentage Yield，含複利 |
| **APR** | 年化報酬率 | Annual Percentage Rate，不含複利 |
| **Yield** | 收益 | 投資產生的回報 |

## 借貸協議

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Lending** | 借出 | 將資產存入協議賺取利息 |
| **Borrowing** | 借款 | 抵押資產借出其他資產 |
| **Collateral** | 抵押品 | 借款時鎖定的資產 |
| **Collateral Ratio** | 抵押率 | 抵押品價值 / 借款價值的比例 |
| **Liquidation** | 清算 | 抵押不足時自動賣出抵押品還債 |
| **Liquidation Price** | 清算價格 | 觸發清算的價格水平 |
| **Health Factor** | 健康因子 | 借貸協議中的安全指標，低於 1 會被清算 |
| **Over-collateralization** | 超額抵押 | 抵押品價值高於借款價值 |
| **Flash Loan** | 閃電貸 | 無需抵押，同一區塊內借還的貸款 |

## 流動性與收益

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Yield Farming** | 收益農場 | 將資產投入協議賺取獎勵 |
| **Liquidity Mining** | 流動性挖礦 | 提供流動性獲得代幣獎勵 |
| **Vault** | 金庫 | 自動化收益策略的智能合約 |
| **Aggregator** | 聚合器 | 整合多個協議尋找最佳收益 |
| **Compounding** | 複利 | 將收益再投資產生更多收益 |
| **Auto-compound** | 自動複利 | 自動將收益再投資 |

## DEX 與交易

| 英文 | 中文 | 解釋 |
|------|------|------|
| **DEX** | 去中心化交易所 | 點對點交易，無需信任 |
| **AMM** | 自動做市商 | 自動定價的交易機制 |
| **CFMM** | 常數函數做市商 | Constant Function Market Maker |
| **x*y=k** | 乘積定常公式 | Uniswap 使用的定價公式 |
| **Concentrated Liquidity** | 集中流動性 | 在特定價格區間提供流動性 |
| **Range Order** | 區間訂單 | 類似限價單的流動性提供方式 |

## 穩定幣

| 英文 | 中文 | 解釋 |
|------|------|------|
| **Stablecoin** | 穩定幣 | 錨定法幣（通常是 USD）的加密貨幣 |
| **Fiat-backed** | 法幣抵押型 | 由現實法幣儲備支持（如 USDC）|
| **Crypto-backed** | 加密抵押型 | 由加密資產超額抵押（如 DAI）|
| **Algorithmic** | 演算法型 | 用演算法維持錨定（風險最高）|
| **Peg** | 錨定 | 維持與目標資產（如 USD）的 1:1 匯率 |
| **Depeg** | 脫錨 | 失去與目標資產的錨定 |

## 進階概念

| 英文 | 中文 | 解釋 |
|------|------|------|
| **MEV** | 最大可提取價值 | Miner Extractable Value，區塊生產者的套利機會 |
| **Sandwich Attack** | 三明治攻擊 | 在前後夾擊交易套利 |
| **Front-running** | 搶先交易 | 看到大單後提前交易獲利 |
| **Arbitrage** | 套利 | 在不同市場間利用價差獲利 |
| **Governance** | 治理 | 代幣持有者投票決定協議變更 |
| **Proposal** | 提案 | 建議協議變更的正式文件 |
| **Quorum** | 法定人數 | 投票通過所需的最低參與度 |
