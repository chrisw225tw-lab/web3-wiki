## 這是什麼？

這是一個開源的 Web3 教育資源，包含：

- **概念解釋** — Web3、區塊鏈、DeFi、NFT、DAO 基礎
- **術語百科** — 100+ 個中英對照術語
- **安全指南** — 如何保護資產、識別詐騙
- **實務教學** — 第一個錢包設置

## 適合誰？

- 完全新手，想了解 Web3 是什麼
- 剛入門，想系統學習基礎
- 需要繁中資源的中文使用者

## 術語速查

需要查詢某個詞？去 [術語索引](docs/terms/index.md)。

常用術語：
- **Wallet** — 錢包
- **Private Key** — 私鑰
- **Gas** — 燃料費
- **DeFi** — 去中心化金融
- **NFT** — 非同質化代幣
- **DAO** — 去中心化自治組織
- **FOMO** — 錯失恐懼症
- **Rug Pull** — 捲款跑路
- **DYOR** — 自己做研究

## 風險聲明

⚠️ **本 Wiki 僅供教育用途，不構成投資建議。**

Web3 是高風險領域：
- 你可能損失全部投資
- 詐騙非常普遍
- 監管不確定
- 技術仍在早期

做任何投資決定前，請自己做研究（DYOR）。
